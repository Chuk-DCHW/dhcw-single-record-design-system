Implementation

Breakpoint Map

Typography tokens are split into desktop and mobile scales. Breakpoint tokens define where each scale applies.

Token Value Usage
breakpoint.mobile.max 767px Use mobile scale below this
breakpoint.tablet.min 768px Use desktop scale from here
breakpoint.desktop.min 1024px Full desktop layout and spacing
.NET MAUI — Map *.mobile tokens to the mobile breakpoint; map *.desktop tokens to tablet and desktop breakpoints.

Blazor — CSS media queries reference breakpoint tokens directly. Use CSS custom properties or token-aware utility classes to switch token sets at the declared breakpoints:
@media (min-width: 768px) { /* apply desktop tokens */ }
